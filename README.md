# City App Web

Application web React pour tester la connexion avec l'API City App.

## Installation

```bash
npm install
```

## Démarrage

1. Assurez-vous que l'API est lancée sur le port 3000
2. Lancez l'application web :

```bash
npm run dev
```

3. Ouvrez votre navigateur sur http://localhost:5173

## Fonctionnalités

- Test de connexion à l'API
- Affichage des villes récupérées
- Interface moderne et responsive

## Structure

- `src/App.jsx` - Composant principal
- `src/main.jsx` - Point d'entrée de l'application
- `vite.config.js` - Configuration Vite avec proxy API
